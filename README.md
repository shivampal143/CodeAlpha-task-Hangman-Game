# CodeAlpha-task-Hangman-Game

Simple Command-Line Hangman Game
A lightweight, text-based Hangman game built in Python. This project is a minimalist implementation of the classic word-guessing game, designed to demonstrate fundamental programming concepts.

Features
Simplified Scope: Uses a built-in list of predefined words (no external APIs or file dependencies required).

Classic Rules: Players get exactly 6 incorrect guesses before the game ends.

Input Validation: Prevents penalties for duplicate guesses or invalid characters (numbers, symbols).

Console-Based: Clean and simple text interface without any heavy graphics or external audio libraries.

Concepts Demonstrated
Random word selection (random module)

Game state management (while loops and if-else logic)

String manipulation and list filtering
